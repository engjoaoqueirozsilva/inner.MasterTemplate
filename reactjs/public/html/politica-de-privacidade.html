Política de Privacidade

Última atualização: 2026

1. Introdução

O In-Set PRO respeita a sua privacidade e está comprometido com a proteção dos dados pessoais, em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018).

2. Dados Coletados

Podemos coletar os seguintes dados:

  • Nome e informações de cadastro
  • Dados de atletas cadastrados
  • Informações de uso da plataforma
  • Dados técnicos (IP, navegador, dispositivo)

3. Finalidade do Uso

Os dados são utilizados para:

  • Operação da plataforma
  • Melhoria da experiência do usuário
  • Geração de análises de performance
  • Segurança e prevenção de fraudes

4. Compartilhamento

Os dados não são vendidos. Podem ser compartilhados apenas com serviços necessários para funcionamento do sistema (ex: hospedagem, e-mail).

5. Armazenamento

Os dados são armazenados em ambiente seguro e protegidos contra acesso não autorizado.

6. Direitos do Usuário

Você pode solicitar:

  • Acesso aos seus dados
  • Correção de dados
  • Exclusão dos dados
  • Revogação de consentimento

7. Cookies

Utilizamos cookies para melhorar a experiência do usuário. Você pode gerenciar isso no navegador.

8. Segurança

Adotamos medidas técnicas para proteger os dados, mas nenhum sistema é 100% seguro.

9. Alterações

Esta política pode ser atualizada periodicamente.

10. Contato

Para dúvidas ou solicitações, entre em contato: contato@inset-pro.com
